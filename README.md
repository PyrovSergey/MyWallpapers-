# MyWallpapers

Application for android allowing to find, view, download and install wallpapers

### Used Libraries
- SwiftyJSON
- Alamofire
- SDWebImage
- ReachabilitySwift
- ChameleonFramework/Swift
 
### Installation
- Requires a key to work with Api pexels.com
